# Images Directory

## og-image.jpg

Please add an Open Graph image here with the following specifications:

- **Filename:** `og-image.jpg`
- **Dimensions:** 1200 x 630 pixels
- **Format:** JPG or PNG
- **Size:** < 1MB recommended

This image will be displayed when your website is shared on social media platforms (Facebook, Twitter, LinkedIn, etc.).

### Creating Your OG Image

1. Use design tools like Canva, Figma, or Photoshop
2. Include your bot name and key message
3. Keep text readable and avoid small fonts
4. Use high contrast colors
5. Test how it looks on different platforms

### Quick Alternative

Until you create a custom image, you can use a placeholder:
- Download a free stock image from Unsplash or Pexels
- Add your bot name using an online image editor
- Save as `og-image.jpg` in this directory
